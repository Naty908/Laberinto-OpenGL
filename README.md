# 🌸 KomorebiGL

OpenGL-based graphics project.

---

## ✨ Description

This project implements graphical rendering using OpenGL, including shaders, model handling, and multimedia resources.

---

## 🚀 Features

* 🎨 3D rendering
* 💡 Shader usage
* 🎵 Audio integration
* 📦 Asset management (textures, models)

---

## 📂 Project Structure

```bash
Komorebi/
├── src/
├── include/
├── assets/
├── shaders/
├── main.cpp
```

---

## ⚙️ Installation & Run

```bash
mkdir build
cd build
cmake ..
make
./app
```
