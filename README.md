# ✧ Laberinto OpenGL

Proyecto desarrollado en **C++** enfocado en la construcción de un entorno interactivo tipo laberinto mediante **OpenGL**, combinando lógica espacial, simulación visual y navegación dentro de un escenario pseudo 3D.  

Más allá de crear únicamente un juego, este proyecto busca explorar la relación entre programación, diseño visual y experiencia interactiva dentro de un entorno gráfico estructurado. ✦

# ✦ Características del Proyecto

### ◌ Exploración Interactiva
Sistema de movimiento dentro del laberinto que permite recorrer distintos caminos y explorar el entorno desde una perspectiva dinámica.

### ◌ Construcción del Laberinto
Generación lógica de paredes, caminos y distribución espacial utilizando coordenadas y transformaciones geométricas.

### ◌ Sistema de Cámara
Implementación de movimiento de cámara para aportar profundidad, orientación y una experiencia de navegación más inmersiva.

### ◌ Simulación Visual 3D
Uso de escalados, rotaciones, traslaciones y proyecciones para crear una representación visual tipo 3D dentro de OpenGL.

### ◌ Progresión de Dificultad
Sistema de niveles diseñado para aumentar gradualmente la complejidad del recorrido.
